# FIRING STICKERS

> 本游戏由人工智能生成，能工智人@Zpolatur 负责策划和 Prompt

一款 2D 俯视角多人射击游戏：单人训练、本地多人、以及服务器权威的联机对战（PvP）。
包含多种玩法模式（死斗 / 团队死斗 / 双排死斗 / 排位提现 Cashout / 生化危机）、海量枪械与近战武器、
可切换的角色皮肤、人机陪练，以及实时公告等社区功能。

## 运行方式

```bash
npm install

# 本地联机模拟（纯前端，file:// 直接打开也可）
npm run dev          # Vite 开发服务器
npm run play         # 构建并以静态方式打开游玩

# 需要联网 / 公告 / 服务器权威对战时，另开一个终端启动 Node 服务器
npm run server       # 中转联机服务器（默认 :8080）
npm start            # 生产服务器（含公告 API 与 /admin 管理页）
```

服务器启动后，前端通过 Vite 代理自动连接 `localhost:8080` 的 `/api` 与 `/admin`；
`/admin`（默认密码 `admin`，可用环境变量 `ANNOUNCE_TOKEN` 修改）用于发布全服公告。

## 操作说明（游戏内按 H 随时查看）

- 移动：`WASD` ／ 触屏摇杆
- 瞄准 / 射击：鼠标 ／ 触屏
- 技能：`Q` `E` `Shift` 等（随武器/皮肤而定）
- 复活队友：长按 `V` ／ `F`
- 暂停 / 设置：`P` 或 `Esc`
- 桌面端鼠标锁定：点画面锁定鼠标，按 `U` 或 `Esc` 显示光标
- 操作说明：`H`

## 更新日志

### Beta 0.6.0

相对上一提交版本（v0.5.x）的全部改动如下。

#### 🆕 新功能
- **服务器权威联机（PvP）**：新增由服务器统一模拟的对战模式，玩家输入上传服务器、由服务器广播实时画面，杜绝客户端作弊与不同步。
- **大厅整合**：移除独立的大厅界面（`LobbyScreen`），进入流程简化为「主菜单 → 装备选择 → 战斗」。
- **实时公告系统**：主菜单右上角 `☰` 菜单可查看管理员发布的公告；基于 SSE 实时推送，在线玩家秒级刷新。
- **在线人数显示**：主菜单实时显示当前在线玩家数。
- **双排死斗网络匹配**：在「双排死斗」模式下可勾选“网络联机匹配”，自动匹配一名真人队友（2 人小队对抗）。
- **新角色皮肤**：👽 外星人（+6% 移速）、🐒 猴子（+18 血量）、🎩 大亨（+5% 射速），并带有对应头部建模。
- **排位提现模式（Cashout）**：3v3v3v3 提现争夺战——夺取现金盒、搬运至提现点提现，含 8 分钟时限与加时赛，带队伍资产排行与队友复活（长按 `V`/`F`）。
- **生化危机新怪物**：🕷️ 爬虫（高速群拥）、🟢 毒爆体（死亡炸出毒云）、📢 尖啸者（加速同伴并震慑玩家）、🟩 孢子怪（持续释放滞留毒云）。
- **桌面端鼠标锁定（Pointer Lock）**：游玩时点画面即把鼠标锁定在画面内，按 `U` / `Esc` 显示光标（移动端不启用）。

#### 🎮 玩法 / 平衡
- **枪械库扩充**（`data/guns.json`）：加特林（需预热）、刘易斯机枪、电浆步枪（三连发+穿墙）、斥候（半自动穿透射手步枪）、毒雾喷射机、闪电鞭（命中减速）、双刃（连段+举刀反弹子弹）、突刺长剑（蓄力冲刺）、RPG-7 爆炸范围上调至 360 等。
- **默认配装改为 2 把武器**，可在装备界面自由切换搭配。
- **地图场景扩充**：赛博都市、西部牛仔、幽静丛林、极寒地带等多张主题竞技场，各有独立地形与视觉风格。
- **人机（BOT）强度可独立调节**，AI 刷新频率与帧率解耦（移动端自动限制以保证流畅）。

#### 🐞 Bug 修复
- 修复中转（relay）模式下 `youPid` 恒为 0 导致客人找不到自己角色、画面无法定位的问题。
- 修复联机对手短暂掉线后正确提示“对手掉线，正在等待重连”，并在宽限期内可恢复同一局（而非强制重新匹配）；房间失效时也能正确引导重连。
- 修复团队死斗按 2队4人 / 3队6人 / 4队8人 正确设定击杀目标。
- 修复人机**贴脸 / 冲撞时不开枪**的问题（所有强度统一生效）。

#### 🎨 UI / UX
- **主菜单重做**：单一“开始游戏”入口，背景图不再叠加暗化遮罩，保留“本游戏由人工智能生成”署名。
- **装备预览增强**：角色实时跟随鼠标朝向；武器详情面板新增按近战 / 激光 / 喷射 / 毒雾等分类的参数条（伤害、射速、弹速、击退、弹匣、穿透、连段等）。
- **受击震屏**、**淘汰 / 伤害面板**（倒地横幅 + 逐条伤害统计 + 重生倒计时）、**击杀分数飘字**。
- **移动端优化**：触屏自动隐藏键鼠 HUD、启用屏幕摇杆与按键，并支持自定义手机端键位布局；设置面板支持帧率限制、音量、全屏。

#### 🌐 联网 / 多人
- 新增 `authoritative.mjs` 权威服务器（默认 30Hz 固定步长 + 客户端插值），支持 `find` 快速匹配、`create`/`join` 房间码（4 位）、断线 `rejoin`（15 秒宽限期保留房间）。
- **快照镜像**：墙体破坏、投掷物飞行、炮塔 / 地雷 / 治疗站部署物、技能视觉效果等双向同步，保证双方看到一致的世界状态（含拆墙后的残墙）。
- 新增**击杀确认音**与**死亡音**，帮助玩家即时感知战况。
